Fangen Sie noch heute an, Daten zu bearbeiten!

Das Hinzufügen von Daten ist einfach! Registrieren Sie sich einfach und erstellen Sie Ihr eigenes Profil. Sobald Sie dies getan haben, können Sie mit dem Hinzufügen und Bearbeiten von Daten beginnen! Neue Daten können hinzugefügt werden, indem Sie auf das orangefarbene Pluszeichen klicken, das nach der Anmeldung angezeigt wird. Um bestehende Einträge zu ändern, klicken Sie auf das Stift-Symbol.

Vergessen Sie nicht, Ihren Datensatz zu verknüpfen. Jede Aktivität (Dienst, Projekt, Ereignis) sollte mit einem Akteur (Organisation, Person) verknüpft sein. Häufig können Einträge jedoch auch mit vielen anderen Einträgen verknüpft werden.
