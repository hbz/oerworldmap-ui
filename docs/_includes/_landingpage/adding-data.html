Start editing data today!

Adding data is easy! Just register and create your own profile. As soon as you have done so, you can start adding and editing data! New data can be added by clicking on the orange plus sign, which appear after you logged in. in order to change existing entries, click on the pen-icon.

Don't forget to link your data set. Each activity (service, project, event) should be linked to an actor (organization, person). But often entries can be linked to many other entries as well.
